# Unity 講習会 2024 応用編

## 前提

Unity をインストール済み
任意の IDE がある(Visual studio, Rider など)
入門編を終えている

## 題材

Unityちゃんアスレチックゲームを作る。

## 学ぶこと

* EditorLayout
* Animation
* Physics
* AudioMixer
* PrefabVariant
* ParticleSystem
* UnityPackage
* Decal Projector
* InputSystem
* TextMeshPro
* Timeline
* Cinemachine
* Terrain
* Tree
* Skybox
* DepthCamera
* PostProcessing
* Lightings
* UIToolkit

## ゲーム仕様

Unityちゃんが障害物を避けながらアイテムを回収してくるゲーム

HP が 0 になるとゲームオーバー
